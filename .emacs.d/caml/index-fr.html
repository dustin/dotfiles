Mode emacs pour Ocaml

Ceci est une version légèrement améliorée du mode emacs de la distribution Ocaml. L'ensemble des fichiers se trouve ici et doit être installé dans local-lisp (ou à un autre endroit visible depuis emacs).

Pour installé ce mode il suffit d'inclure le fichier ocaml.emacs de la distribution dans votre fichier de lancement .emacs. Il vous faudra peut-être ajuster le chemin d'accès aux sources si celles-ci ne sont pas installées dans un dossier qui est sur le chemin de d'accès par défaut de Emacs.
